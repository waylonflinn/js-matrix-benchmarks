This page benchmarks the performance of a collection of matrix libraries for numerical computation. Sylvester (v0.1.3, 2015-11-19).
Benchmarks for each library are run in an iframe. Results measure hundreds of iterations per second of the target operation.

Adapted by Waylon Flinn from the benchmarks created by Stephen Bannasch for glmatrix library. JavaScript Plotting library: Flotr; Sortable Table by Andrew Tetlaw.
The source code for these benchmarks can be found here on github. Pull requests welcome!
